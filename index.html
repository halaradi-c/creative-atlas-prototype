            How the lock is enforced

            Each rung is a capability boundary, not a setting. Atlas holds only read + insights scopes; the write scopes that could move spend are not granted at the credential layer — so there is no code path to act, even by accident.

            • Safe-read — observes performance
            • Safe-notify — posts the morning brief
            • Safe-draft — proposes; a human approves
            • Act — guarded auto-pilot (two-key + kill-switch)
