# Creative Atlas+ — Prototype

A clickable static prototype of **Creative Atlas+**, Calo's Performance-Marketing intelligence product.
Pure static HTML/CSS/vanilla-JS. All figures are mock data, badged **Prototype · Preview Data**.

**Live:** see the repo's GitHub Pages URL.

7 screens: Command Center · Creative Intelligence (Creative Merit Score) · Cohort Economics ·
Spend Efficiency (Next-Dollar Map) · Daily Pulse & Alerts · Ask Calo · The Frontier / Roadmap.

Run locally: `python3 -m http.server 8123` then open http://localhost:8123/
